# Задание 1

## Задача 1

### Текст задачи

На основе заданного списка чисел получить список из значений true и false. True
— если соответствующий элемент исходного списка чётный, и false — в
противном случае

### Алгоритм решения

Пользователь вводит числа через пробел, вызывается встроенная функция которая убирает пробелы и преобразовывает 
в массив, дальше вызывается функция Array.toList которая преобразывается части массива в список строк, дальше вызывается функция List.map int которая преобразует строку в число. Далее создаем список через функцию List.map которая использует полученный список чисел, вызвывается функция chet в которой проверяется равен ли остаток от деления на 2 нулю возвращает true или false и создает список из них.

## Задача 2

### Текст задачи

Список содержит строки, обозначающие римские числа от I до IX. Найти в
десятичной системе счисления сумму элементов списка

### Алгоритм решения

Пользователь вводит римские числа через провел, потом мы создаем список удаляя пробелы и преобразовывая массив строк в список, далее мы создаем переменную которая вызывает функция List.fold использующую полученные список римских цифр, вызывается функция addroman c 0 и первым элементом списка, в функции 0 присвается переменная sm и к ней прибавляется число которое приходит из функции romanToInt, в функции римские цифры преобразуются в цифры в десятичной системе счисления, оно возвращается суммируется, и проходит так по всем элементам списка и выводит их сумму.
